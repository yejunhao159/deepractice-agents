# 第13章：智能旅行助手

> 🔄 本章待用 AgentX + PromptX 重写

## 项目目标

构建一个完整的智能旅行助手，展示 AgentX 多智能体协作能力。

## 核心功能

1. **智能行程规划** - 根据用户偏好自动生成行程
2. **地图可视化** - 景点标注和路线绘制
3. **预算计算** - 自动计算各项费用
4. **行程编辑** - 支持实时调整
5. **导出功能** - PDF/图片导出

## 技术架构

- 前端：Vue3 + TypeScript
- 后端：Node.js + AgentX
- 智能体：PromptX 角色定义
- 外部服务：地图 API、LLM API
