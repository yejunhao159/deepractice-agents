## 章节概述

本章聚焦于现代智能体的核心组件——大语言模型(LLM),系统介绍其工作原理、交互技术和应用边界。

**四个核心问题**:

1. **语言模型如何演进?** (3.1节)
   回顾从统计语言模型到神经网络,再到Transformer的发展历程

2. **如何有效交互?** (3.2节)
   掌握提示工程(Prompt Engineering)的设计原则与实践技巧

3. **LLM能做什么?不能做什么?** (3.3节)
   分析大语言模型的核心能力与技术局限

4. **如何构建智能体架构?** (3.4节)
   理解智能体的四层架构设计及模型选择策略

---

## 学习目标

完成本章学习后,读者应能够:
- 理解大语言模型的技术演进脉络
- 设计完整的智能体提示系统
- 识别大语言模型的能力边界
- 规划智能体的架构和工具需求
