# 第四章 智能体经典范式

在上一章中，我们深入探讨了大语言模型的能力与边界，理解了为什么需要通过智能体架构来弥补模型的不足。现在，是时候动手实践了。

本章将带你从零实现三种经典的智能体范式：

| 范式 | 核心思想 | 适用场景 |
|------|---------|---------|
| **ReAct** | 思考与行动交替进行，边想边做 | 需要外部工具、动态调整的任务 |
| **Plan-and-Solve** | 先制定完整计划，再严格执行 | 结构清晰、步骤明确的任务 |
| **Reflection** | 执行后自我反思，迭代优化 | 对质量要求高的任务 |

通过亲手实现这些范式，你将深刻理解智能体的工作原理。这不仅是学习，更是为后续使用 AgentX 等框架打下坚实基础——只有理解了底层机制，才能在框架无法满足需求时进行定制和扩展。
