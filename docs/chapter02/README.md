## 🎯 本章核心问题

为了深刻理解现代智能体为何呈现出如今的形态，以及其核心设计思想的由来，本章将回溯历史：

1. **符号主义时代**：最早的"智能"如何在逻辑与符号的规则体系中被定义？
2. **分布式思想**：从单一、集中的智能模型到分布式、协作式智能思想的重大转折
3. **学习范式**："学习"如何彻底改变了智能体获取能力的方式？
4. **现代爆发**：2023年以来，智能体技术为何迎来爆发式增长？

## 💡 学习建议

1. **理解历史脉络**：每个新范式的出现都是为了解决上一代的核心"痛点"
2. **动手实践**：通过构建ELIZA来体会规则系统的优势与局限
3. **关注最新进展**：2023-2025是智能体发展的关键转折期
