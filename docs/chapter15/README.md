# 第15章：构建赛博小镇

> 🔄 本章待用 AgentX 多智能体系统重写

## 项目目标

构建一个充满生命力的 AI 小镇，探索智能体与游戏引擎的结合。

## 核心功能

1. **智能 NPC 对话系统** - 自然语言对话，角色化回应
2. **记忆系统** - 短期记忆 + 长期记忆
3. **好感度系统** - 动态关系演变
4. **游戏化交互** - 2D 像素风格场景
5. **实时日志** - 对话记录与分析

## 技术架构

- 游戏引擎：Godot / Web Canvas
- 后端：Node.js + AgentX
- 智能体：每个 NPC 是独立的 AgentX Agent
- 记忆：PromptX Engram 记忆系统
